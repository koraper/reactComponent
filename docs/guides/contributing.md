# 기여하기

프로젝트에 기여해주셔서 감사합니다!

## 개발 환경 설정

1. 저장소 클론
```bash
git clone <repository-url>
cd reactComponent
```

2. 의존성 설치
```bash
npm install
```

3. Storybook 실행
```bash
npm run dev
```

## 새 컴포넌트 추가

자동 생성 스크립트를 사용하세요:

```bash
npm run generate:component ComponentName [category]
```

예시:
```bash
npm run generate:component MyButton atoms
```

## 코딩 스타일

- ESLint와 Prettier를 사용합니다
- TypeScript를 사용합니다
- 각 컴포넌트는 테스트와 Storybook 스토리를 포함해야 합니다
